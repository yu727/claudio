## 1. 封面 ↔ 歌词 3D 翻转

### 设计文档要求
- 默认: 专辑封面 + 动态取色背景
- 点击封面: CSS 3D `rotateY(180deg)` 过渡 0.6s
- 翻出: 歌词面板，逐行高亮 + 自动滚动
- 再次点击: 翻转回封面
- 实现: CSS `perspective` + `transform-style: preserve-3d` 容器正反两面

### 当前代码状态
- PlayerPage.tsx 已有 `coverFlipped` state 但未使用
- KaraokeLyrics 组件已存在（显示歌词）
- 但没有翻转容器和翻转逻辑

### 实现方案

修改 `apps/web/src/pages/PlayerPage.tsx`:

在 player-lower 区域添加翻转容器:

```tsx
<div className="cover-flip-container" style={{ perspective: '1000px' }}>
  <div 
    className={`cover-flipper ${coverFlipped ? 'flipped' : ''}`}
    onClick={() => setCoverFlipped(!coverFlipped)}
    style={{
      transformStyle: 'preserve-3d',
      transition: 'transform 0.6s cubic-bezier(0.4, 0, 0.2, 1)',
      transform: coverFlipped ? 'rotateY(180deg)' : 'rotateY(0)',
    }}
  >
    {/* 正面: 歌曲信息 + 控制区 */}
    <div className="cover-front" style={{ backfaceVisibility: 'hidden' }}>
      {/* 现有的 song-title, song-artist, progress, controls */}
    </div>
    
    {/* 反面: 歌词 */}
    <div className="cover-back" style={{ 
      backfaceVisibility: 'hidden',
      transform: 'rotateY(180deg)',
      position: 'absolute',
      inset: 0,
    }}>
      <KaraokeLyrics songId={nowPlaying?.songId} currentTimeMs={progressMs} />
    </div>
  </div>
</div>
```

添加 CSS 到 global.css:
```css
.cover-flip-container {
  perspective: 1000px;
  width: 100%;
}
.cover-flipper {
  transform-style: preserve-3d;
  transition: transform 0.6s cubic-bezier(0.4, 0, 0.2, 1);
  position: relative;
  cursor: pointer;
}
.cover-flipper.flipped {
  transform: rotateY(180deg);
}
.cover-front, .cover-back {
  backface-visibility: hidden;
}
.cover-back {
  transform: rotateY(180deg);
  position: absolute;
  inset: 0;
  overflow-y: auto;
}
```

**重要**: 翻转容器需要包含的是歌曲信息区（title/artist/progress/controls），不包含 dj-header 和 AudioSpectrum。把现有的 player-lower 拆分为两部分：
- DJ 头部 + 频谱 → 始终可见
- 歌曲信息 + 控制 → 可翻转

### 翻转触发
- 点击 player-lower 区域的歌曲信息部分触发翻转
- 不需要单独的封面图（因为当前设计没有大封面图，是 FM 风格）
