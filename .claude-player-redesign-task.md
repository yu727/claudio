# Claudio 播放器页面重设计 — 网易云风格

## 当前问题

1. **歌词通过翻转卡片显示** — 点击封面区域会翻转到背面显示歌词，非常不直觉
2. **封面和歌词没有并排显示** — 网易云的经典布局是左边大封面+右边滚动歌词
3. **封面只有 180x180** — 太小了，应该作为主视觉
4. **控制区和歌曲信息混在一起** — 没有清晰的层次

## 目标设计（参考网易云音乐播放器）

### 整体布局（从上到下）

```
┌─────────────────────────────────────────────┐
│  [DJ header: 🎧 CLAUDIO | status | time]    │  ← 保留现有 DJ 状态栏
│  [AudioSpectrum 波形条]                      │  ← 保留
├──────────────────────┬──────────────────────┤
│                      │                      │
│    ┌──────────────┐  │   🎵 歌曲标题         │
│    │              │  │   歌手名              │
│    │   专辑封面    │  │                      │
│    │   (大图)      │  │   [当前歌词行 高亮]    │
│    │  300x300+    │  │   下一行歌词           │
│    │              │  │   下一行歌词           │
│    │  ↻ 旋转动画  │  │   下一行歌词           │
│    │              │  │   下一行歌词           │
│    └──────────────┘  │   ...（可滚动）        │
│                      │                      │
├──────────────────────┴──────────────────────┤
│  0:13 ═══════════●═══════════════════ 3:45  │  ← 进度条
│     🔀  ⏮  ▶/⏸  ⏭  🔁   🔊 ═══●═       │  ← 控制栏
├─────────────────────────────────────────────┤
│  [WaveformBar 波形]                         │
├─────────────────────────────────────────────┤
│  [ChatArea + IntentInput]                   │
│  [Search toggle]                            │

└─────────────────────────────────────────────┘
```

### 关键设计要求

1. **去掉 cover-flipper 翻转机制** — 完全删除 `.cover-flip-container` / `.cover-flipper` / `.cover-front` / `.cover-back` 这套翻转逻辑
2. **封面和歌词并排** — flex 布局，左边封面（约 45% 宽），右边歌词（约 55% 宽）
3. **大封面** — 280px-320px，圆角 16px，播放时有 CSS 旋转动画（唱片效果，`animation: spin 20s linear infinite`，暂停时 `animation-play-state: paused`）
4. **封面光晕** — 保留现有的 cover-glow 效果
5. **歌词面板** — 直接放在封面右边，当前行高亮 + 放大，其他行半透明，可滚动
6. **歌曲标题和歌手** — 放在歌词面板顶部
7. **进度条和控制区** — 跨越整个底部，不跟封面/歌词挤在一起
8. **收藏按钮** — 放在标题旁边（保留现有逻辑）
9. **错误横幅** — 保留（在进度条上方）
10. **"Tap to Play" 横幅** — 保留

### 响应式

- 移动端（<768px）：封面和歌词改为上下堆叠（flex-direction: column），封面居中
- 桌面端：左右并排

### CSS 变化要点

- 新增 `.player-content` flex 容器（封面区 + 歌词区）
- 新增 `.cover-section` — 封面容器，包含大封面图
- 新增 `.cover-disc` — 唱片旋转效果容器
- 新增 `.lyrics-section` — 右侧歌词区，固定高度，overflow-y auto
- 删除 `.cover-flip-container` / `.cover-flipper` / `.cover-front` / `.cover-back` 相关 CSS
- 删除 `coverFlipped` state 和相关 click handler
- `.player-lower` 改为新的布局结构
- 响应式 media query

### KaraokeLyrics 组件

- 不需要修改 KaraokeLyrics 组件本身，它已经是自包含的
- 只需要改变它在 PlayerPage 中的放置位置（从 cover-back 移到 lyrics-section）
- 歌词面板需要一个固定高度（比如 calc(100vh - 300px) 或类似值），让歌词可滚动

### 不要改动的部分

- PlayerPage 的所有状态管理（store hooks, useState 等）
- AudioVisualizer / FluidBlobs / BorderGlow / SpectrumBars
- ChatArea / IntentInput / SearchPanel / DjPanel / UserPanel
- WaveformBar
- 进度条拖拽逻辑
- 播放控制逻辑
- 所有 API 调用
- WebSocket 连接
